x2c_Clarke_MCHP

MCHP Clark Transform

Description

Moves a three-axis, two-dimensional coordinate system, referenced to the stator, onto a two-axis system, keeping the same reference.

Inports

  • a: A phase
  • b: B phase

Outports

  • alpha: Shifted A phase
  • beta: Shifted B phase

Mask Parameters

None

Implementations

  • FiP16: 16 Bit Fixed Point
  • FiP32: 32 Bit Fixed Point
  • Float32: 32 Bit Floating Point
