x2c_PerturbationGen

X2C Perturbation Generator

Description

Generation of a square wave with amplitude (A) and frequency (f) with different slew rates..

Inports

  • f: Frequency
  • dc: Duty Cycle
  • S_in: Signal in

Outports

  • u: Square wave output
  • S_out: Signal out (= Signal in)

Mask Parameters

  • fmax: Maximum Frequency in Hz
  • LL: Output Low Level
  • LH: Output High Level
  • Tr: Rising time in seconds. Slew rate will be 1/Tr.
  • Tf: Falling time in seconds. Slew rate will be 1/Tf.
  • ts_fact: Multiplication factor of base sampling time (in integer format).

Implementations

  • FiP16: 16 Bit Fixed Point
  • FiP32: 32 Bit Fixed Point
  • Float32: 32 Bit Floating Point
  • Float64: 64 Bit Floating Point
