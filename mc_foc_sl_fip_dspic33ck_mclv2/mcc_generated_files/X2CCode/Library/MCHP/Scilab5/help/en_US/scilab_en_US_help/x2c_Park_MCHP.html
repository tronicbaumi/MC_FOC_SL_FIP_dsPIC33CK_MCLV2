x2c_Park_MCHP

MCHP Park Transform

Description

The two-axis orthogonal system with the axis called alpha-beta transform into another two-axis system that is rotating with the rotor flux. Two-axis rotating coordinate system is called the d-q axis. Theta represents the rotor angle.

Inports

  • alpha:
  • beta:
  • theta:

Outports

  • d:
  • q:

Mask Parameters

None

Implementations

  • FiP16: 16 Bit Fixed Point
  • FiP32: Fixed Point 32 Bit implementation
  • Float32: 32 Bit Floating Point
