x2c_RandomGen

X2C Ramdom Generator

Description

Generation of white noise with amplitude (A)

Inports

  • S_in: Signal in

Outports

  • u: Output for noise + input signal
  • S_out: Signal out (= Signal in)

Mask Parameters

  • A: Amplitude of the Random Generator
  • ts_fact: Multiplication factor of base sampling time (in integer format).
  • D: Generate a new value every D samples (1, 2, 3,...)

Implementations

  • FiP16: 16 Bit Fixed Point
  • FiP32: 32 Bit Fixed Point
  • Float32: 32 Bit Floating Point
  • Float64: 64 Bit Floating Point
