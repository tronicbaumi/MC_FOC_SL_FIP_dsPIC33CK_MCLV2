Calculation Floating Point Implementation:
\begin{equation*}
	\mathrm{Out}
		=
		\mathrm{e}^{\mathrm{In}}
\end{equation*}

Calculation Fixed Point Implementation:
\begin{equation*}
	\mathrm{Out}
		=
	\begin{cases}
		\mathrm{e}^{\mathrm{In}} & \mathrm{In} \le 0
		\\
		1 & \mathrm{In} > 0
	\end{cases}
\end{equation*}
